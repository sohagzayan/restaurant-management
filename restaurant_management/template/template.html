Welcome to the Restaurant Management API

This API allows you to manage restaurants, their locations, and menus, along with custom user authentication. Follow the routes below to explore the available functionality.

Available API Endpoints

  • POST /api/auth/register/ - Register a new user
  • GET /api/auth/userlist/ - Get all user list
  • POST /api/auth/login/ - User login
  • PUT /api/auth/change_password/ - Change user password
  • POST /api/auth/logout/ - Logout user
  • POST /api/restaurants-create/ - Create new restaurant
  • GET /api/restaurants-list/ - Get all restaurants
  • POST /api/restaurants/1/locations-create/ - Create location
  • POST /api/locations/1/menus-create/ - Create menu
  • GET /api/locations/1/menus-list/ - Get menu list
  • POST /api/menus/1/categories-create/ - Create category
  • GET /api/menus/1/categories-list/ - Get category list
  • POST /api/categories/1/items-create/ - Create item
  • GET /api/categories/1/items-list/ - Get item list
